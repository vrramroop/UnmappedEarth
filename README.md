# UnmappedEarth
A realistic adventure game based on Earth (the one we live on) about being able to capture fish, birds, and animals (water, air, and land) for the purpose of a bestiary. The content is all realistic, for example a fish that can be caught, say in the Canadian  Arctic is an Arctic Char, unlocking the Arctic bestiary and logging an Arctic Char. There is also lore (adding soon).
